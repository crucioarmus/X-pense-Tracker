# XPense Tracker

XPense Tracker is a **C++ command-line personal finance management tool** designed to help users track expenses, manage loans between friends, split bills, and plan financial goals.

This project demonstrates **modular programming, data handling, and real-life financial logic** using C++.

---

# Features

### Loan Manager
Track money you lend to friends and repayments.

- Record money lent to friends
- Track who owes you and how much
- Handle partial repayments
- Automatically remove loan records when fully paid
- Loan sources tracked from:
  - Savings
  - Fun quota
  - Core expenses

---

### Goal Planner
Helps users estimate how long it will take to reach a financial goal.

- Enter monthly income
- Add multiple expenses
- Calculate remaining money
- Estimate months required to achieve a savings goal

---

### Bill Splitter
Quickly split shared expenses among friends.

- Input total bill
- Enter number of participants
- Calculate individual share
- Automatically deduct your share from:
  - Fun quota
  - Savings (if fun quota is insufficient)

---

### Daily Expense Tracker
Simple monthly tracker for daily spending habits.

- Record small daily savings
- Track fun spending
- Monitor remaining budget
- Move unused fun quota to savings at the end of the month

---

### Balance Overview
Quick view of your financial status.

Displays:
- Savings
- Fun quota
- Core expenses
